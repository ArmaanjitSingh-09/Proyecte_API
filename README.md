# Proyecte_API - Gestor de Jugadors de Futbol

Gestor de jugadors de futbol desenvolupat en Python amb emmagatzematge en fitxer JSON.  
Projecte de Programació - SMX1B.

## Funcionalitats

- Afegir nous jugadors
- Buscar jugador (amb obertura automàtica de la imatge)
- Llistar tots els jugadors
- Eliminar jugadors
- Menú interactiu per terminal

## Tecnologies utilitzades

- Python 3
- JSON (persistència de dades)
- Mòdul webbrowser
